# Social Sprouts — Autism-Friendly AI Communication Platform

## Overview
Social Sprouts is an autism-friendly communication and support website designed to help neurodivergent users improve social interaction, manage daily routines, and access educational resources in a calm and accessible environment.

The platform provides a simple, high-contrast, and user-friendly interface tailored for individuals with autism spectrum disorder (ASD).

---

## Features

### 🤖 AI Chatbot
- Autism-friendly conversational assistant
- Provides communication support
- Encourages comfortable interaction
- Simple and easy-to-understand responses

### 🧠 Social Skills Training
- Helps improve communication abilities
- Supports social interaction practice
- Builds confidence in conversations

### 🎥 Educational Videos
- Autism-friendly learning materials
- Interactive and visual educational content
- Easy-to-follow resources

### 📅 Task Scheduler
- Helps users organize daily activities
- Supports routine management
- Improves productivity and planning

---

## User Interface Highlights
- Dark theme with high-contrast neon colors
- Minimal and distraction-free design
- Large readable text and cards
- Accessibility-focused layout
